Projet basé sur la formation « Spring Framework : mise en œuvre » animée par Étienne LANGLET chez ENI Service.

## Technologies
- Java 8
- Spring MVC (XML)
- JSP
- Apache Tomcat 9
- Maven
- PostgreSQL

## Lancer le projet
1. Importer le projet dans IntelliJ
2. Configurer Tomcat 9
3. Accéder à http://localhost:8081/banque-web/home
